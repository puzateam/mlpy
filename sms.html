  • ประกันคุณภาพภายใน
  • มาตรฐานการศึกษา (สมศ.)
  • ผลงานเด่น
  • กลับเว็บหลัก

ระบบการประกันคุณภาพภายใน (IQA)

รายงานการประเมินตนเอง (SAR)

สถานศึกษามีการจัดทำรายงานการประเมินตนเองเป็นประจำทุกปี เพื่อสะท้อนผลการดำเนินงานและวางแผนพัฒนาคุณภาพอย่างต่อเนื่อง

  • SAR ปีการศึกษาปัจจุบัน PDF

แผนพัฒนาคุณภาพการศึกษา

แผนงานที่แสดงทิศทางการพัฒนาคุณภาพของสถานศึกษาในระยะต่างๆ สอดคล้องกับนโยบายและบริบทของสถานศึกษา

  • แผนปฏิบัติการประจำปี PDF

มาตรฐานการศึกษาเพื่อการประเมินคุณภาพภายนอก (สมศ.)

มาตรฐานที่ 1: คุณภาพของผู้เรียน

1.1 ผลสัมฤทธิ์ทางวิชาการของผู้เรียน

  • ความสามารถในการอ่าน การเขียน การสื่อสาร และการคิดคำนวณ
  • ความสามารถในการคิดวิเคราะห์ คิดอย่างมีวิจารณญาณ อภิปราย แลกเปลี่ยนความคิดเห็น และแก้ปัญหา
  • ความสามารถในการสร้างนวัตกรรม
  • ความสามารถในการใช้เทคโนโลยีสารสนเทศและการสื่อสาร
  • ผลสัมฤทธิ์ทางการเรียนตามหลักสูตรสถานศึกษา
  • ความรู้ ทักษะพื้นฐาน และเจตคติที่ดีต่องานอาชีพ
คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • รายงานผลสัมฤทธิ์ทางการเรียน PDF
  • แฟ้มสะสมผลงานนักเรียน PDF
  • ผลงาน/โครงงาน/นวัตกรรมของนักเรียน PDF

1.2 คุณลักษณะที่พึงประสงค์ของผู้เรียน

  • การมีคุณธรรม จริยธรรม ค่านิยมที่พึงประสงค์
  • ความภูมิใจในท้องถิ่นและความเป็นไทย
  • การยอมรับที่จะอยู่ร่วมกันบนความแตกต่างและหลากหลาย
  • สุขภาวะทางร่างกาย และจิตสังคม
คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • บันทึกพฤติกรรมนักเรียน PDF
  • รายงานโครงการ/กิจกรรมส่งเสริมคุณธรรม PDF
  • รายงานกิจกรรมส่งเสริมสุขภาพ PDF

มาตรฐานที่ 2: กระบวนการบริหารและการจัดการ

2.1 การมีเป้าหมาย วิสัยทัศน์ และพันธกิจที่สถานศึกษากำหนดชัดเจน

สถานศึกษามีการกำหนดเป้าหมาย วิสัยทัศน์ และพันธกิจที่ชัดเจน สอดคล้องกับบริบทของสถานศึกษาและความต้องการของชุมชน สื่อสารให้ผู้เกี่ยวข้องทุกฝ่ายรับทราบและเข้าใจตรงกัน

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • เอกสารวิสัยทัศน์ พันธกิจ เป้าประสงค์ของสถานศึกษา PDF
  • รายงานการประชุมคณะกรรมการสถานศึกษา/ผู้เกี่ยวข้อง PDF

2.2 การมีระบบบริหารจัดการคุณภาพของสถานศึกษา

สถานศึกษามีระบบการบริหารจัดการคุณภาพที่ชัดเจนและดำเนินการตามระบบอย่างต่อเนื่อง มีการนำข้อมูลมาใช้ในการปรับปรุงและพัฒนาคุณภาพสถานศึกษา

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • คู่มือระบบบริหารจัดการคุณภาพของสถานศึกษา PDF
  • แผนพัฒนาคุณภาพจากผล SAR PDF

2.3 การดำเนินงานพัฒนาวิชาการที่เน้นคุณภาพผู้เรียนรอบด้าน

สถานศึกษามีการพัฒนาหลักสูตรสถานศึกษาให้สอดคล้องกับหลักสูตรแกนกลางและความต้องการของผู้เรียนและท้องถิ่น ส่งเสริมการจัดการเรียนรู้ที่หลากหลายและเน้นผู้เรียนเป็นสำคัญ

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • หลักสูตรสถานศึกษา PDF
  • รายงานโครงการพัฒนาวิชาการ PDF

2.4 การพัฒนาครูและบุคลากรให้มีความเชี่ยวชาญทางวิชาชีพ

สถานศึกษาส่งเสริมและสนับสนุนให้ครูและบุคลากรได้รับการพัฒนาตนเองอย่างต่อเนื่อง ทั้งด้านความรู้ ความสามารถ และทักษะในการจัดการเรียนรู้และปฏิบัติงาน

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • แผนพัฒนาครูและบุคลากร PDF
  • รายงานการอบรม/สัมมนาของครู PDF

2.5 การจัดสภาพแวดล้อมทางกายภาพและสังคมที่เอื้อต่อการจัดการเรียนรู้

สถานศึกษาจัดอาคารสถานที่ สภาพแวดล้อม แหล่งเรียนรู้ และสิ่งอำนวยความสะดวกให้เพียงพอ ปลอดภัย สะอาด และเอื้อต่อการเรียนรู้ของผู้เรียน

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • ภาพถ่ายสภาพแวดล้อม/แหล่งเรียนรู้ รูปภาพ
  • แผนความปลอดภัยในสถานศึกษา PDF

2.6 การจัดระบบเทคโนโลยีสารสนเทศเพื่อสนับสนุนการบริหารจัดการและการเรียนรู้

สถานศึกษามีการนำเทคโนโลยีสารสนเทศมาใช้ในการบริหารจัดการและสนับสนุนการจัดการเรียนการสอนอย่างเหมาะสมและมีประสิทธิภาพ

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • แผนการใช้ ICT ของสถานศึกษา PDF
  • ข้อมูลระบบ/แพลตฟอร์มการเรียนรู้ที่ใช้ เอกสาร

มาตรฐานที่ 3: กระบวนการจัดการเรียนการสอนที่เน้นผู้เรียนเป็นสำคัญ

3.1 การจัดการเรียนรู้ผ่านกระบวนการคิดและปฏิบัติจริง และนำไปประยุกต์ใช้ในชีวิตได้

ครูจัดการเรียนการสอนที่ส่งเสริมให้ผู้เรียนได้คิดวิเคราะห์ แก้ปัญหา เรียนรู้จากการปฏิบัติจริง และเชื่อมโยงความรู้สู่การนำไปใช้ในชีวิตประจำวัน

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • แผนการจัดการเรียนรู้ที่เน้น Active Learning PDF
  • ตัวอย่างผลงาน/ชิ้นงานนักเรียนจากการปฏิบัติ PDF/รูปภาพ

3.2 การใช้สื่อ เทคโนโลยีสารสนเทศ และแหล่งเรียนรู้ที่เอื้อต่อการเรียนรู้

ครูเลือกใช้สื่อ นวัตกรรม เทคโนโลยี และแหล่งเรียนรู้ทั้งภายในและภายนอกสถานศึกษามาประกอบการจัดการเรียนรู้ได้อย่างเหมาะสมและหลากหลาย

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • บันทึกการใช้สื่อ/เทคโนโลยีในการสอน เอกสาร
  • รายการแหล่งเรียนรู้ที่ใช้ เอกสาร

3.3 การมีการบริหารจัดการชั้นเรียนเชิงบวก

ครูสร้างบรรยากาศในชั้นเรียนที่อบอุ่น ปลอดภัย ส่งเสริมการเรียนรู้ และให้ผู้เรียนมีส่วนร่วมในการเรียนอย่างมีความสุข

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • บันทึกการสังเกตชั้นเรียน เอกสาร
  • ผลสำรวจความพึงพอใจของนักเรียนต่อบรรยากาศในชั้นเรียน เอกสาร

3.4 การตรวจสอบและประเมินผู้เรียนอย่างเป็นระบบ และนำผลมาพัฒนาผู้เรียน

ครูมีวิธีการวัดและประเมินผลที่หลากหลาย สอดคล้องกับสิ่งที่ต้องการวัด และนำผลการประเมินมาใช้ในการวินิจฉัย ปรับปรุงการสอน และพัฒนาผู้เรียนเป็นรายบุคคล

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • ตัวอย่างเครื่องมือวัดและประเมินผล PDF
  • บันทึกการสอนซ่อมเสริม/พัฒนาผู้เรียน เอกสาร

3.5 การมีการแลกเปลี่ยนเรียนรู้และให้ข้อมูลสะท้อนกลับเพื่อพัฒนาและปรับปรุงการจัดการเรียนรู้

ครูและผู้เกี่ยวข้องมีการแลกเปลี่ยนเรียนรู้ (PLC) และให้ข้อมูลป้อนกลับซึ่งกันและกันเพื่อนำไปพัฒนาการจัดการเรียนการสอนอย่างต่อเนื่อง

คลิกเพื่อดูตัวอย่างหลักฐาน/ร่องรอย
  • รายงานการประชุม/กิจกรรม PLC PDF
  • แบบบันทึกการสังเกตการสอนและให้ข้อมูลย้อนกลับ เอกสาร
